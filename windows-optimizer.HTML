Windows 11 Optimization Commands


Before You Start

1. Press Windows + S
2. Type: Create a restore point
3. Open it
4. Click Create
5. Name it: Before Optimization

Always run commands as Administrator

System Repair & Health

1. Press Windows + X
2. Click Terminal (Admin)
3. Paste command
4. Press Enter
5. Wait until finished
6. Restart PC

sfc /scannow
DISM /Online /Cleanup-Image /RestoreHealth
chkdsk /f /r

Performance Optimization

1. Open Terminal (Admin)
2. Paste command
3. Press Enter
4. Restart PC if required

powercfg -duplicatescheme e9a42b02-d5df-448d-aa00-03f14749eb61
powercfg /setactive e9a42b02-d5df-448d-aa00-03f14749eb61
powercfg /h off
wmic computersystem where name="%computername%" set AutomaticManagedPagefile=True

Disk Cleanup

1. Press Windows + R
2. Type command
3. Press Enter
4. Select Drive C
5. Check all boxes
6. Click OK

cleanmgr
cleanmgr /sageset:1
cleanmgr /sagerun:1
del /s /q %temp%\*

Network Optimization

1. Open Terminal (Admin)
2. Paste command
3. Press Enter
4. Restart PC if needed

ipconfig /flushdns
netsh int tcp set global autotuninglevel=normal
netsh winsock reset

Gaming Optimization

1. Open Terminal (Admin)
2. Paste command
3. Press Enter
4. Restart PC

bcdedit /set useplatformclock false

Windows Update Reset

1. Open Terminal (Admin)
2. Run stop command
3. Wait
4. Run start command

net stop wuauserv
net start wuauserv

Startup Optimization

1. Open command
2. Go to Startup tab
3. Disable unnecessary apps

msconfig
taskmgr
